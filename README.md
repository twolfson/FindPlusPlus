# FindPlusPlus

Find functionality from Notepad++ for Sublime Text 2.

## Getting Started
Install the module with: `npm install FindPlusPlus`

```javascript
var FindPlusPlus = require('FindPlusPlus');
FindPlusPlus.awesome(); // "awesome"
```
